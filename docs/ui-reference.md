# UI Reference

I want the app to look like a modern food/AI SaaS web app.

## Style direction

- Clean light background
- White cards
- Rounded corners
- Soft shadows
- Green accent color
- Modern top navbar
- Large centered content area
- Responsive card grid
- Professional but simple university-project look

## Pages

### Home
Landing page with hero section, short explanation, and CTA buttons.

### Pantry
Card-based pantry editor with clean grid rows.

### Generate
Centered generation form card with loading state.

### Recipes
Responsive recipe card grid.

### Recipe Detail
Recipe header, nutrition stat cards, ingredients table, instructions card, chatbot edit card.
