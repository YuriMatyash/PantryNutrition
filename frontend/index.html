Pantry Recipe Agent

Cook smarter with PantryAI

Save your pantry, generate AI-powered recipes, calculate USDA nutrition, and edit recipes with a simple chatbot flow.

1) Save Pantry Ingredients

Track what you have using clean units: g, ml, and unit.

2) Generate Recipes with AI

Pick meal preferences and generate practical pantry-based meals.

3) USDA Nutrition Lookup

Estimate calories, protein, carbs, and fat for each recipe.

4) Save & Edit Later

Open saved recipes and refine them using chatbot-style requests.

5) Conversation History

User and assistant recipe messages are stored for traceability.

App Flow

Pantry → AI recipe generation → USDA nutrition → save recipe → edit with chatbot.
